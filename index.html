Can a model read its context from JPEG pages?

We build one task log. We show it to one model two ways. Once as plain text. Once as JPEG pictures of the same text. We ask the same questions both times. Then we score the answers. That is the whole experiment.

No API key set. Chat works in demo mode. The test needs a real key in .env.

How the test works

  1. We make a fake project log. It is deterministic. Same seed, same log.
  2. The log mutates state. Leads change. Budgets move. Old values get replaced.
  3. At checkpoints we ask questions. Who leads now? What was the first access phrase?
  4. One arm reads plain text. One arm reads JPEG pages of the same text.
  5. Same pinned model. Temperature zero. We flip which arm goes first.
  6. Python scores it and draws the charts. You read them.

One run is a pilot. It shows effect size and noise. It does not prove a law.
